# Charte Graphique - EcoRide

Ce document définit l'identité visuelle de l'application EcoRide.

## 1. Palette de Couleurs

L'identité EcoRide repose sur des tons naturels évoquant l'écologie (vert) et l'énergie (jaune).

| Couleur | Code HEX | Usage Principal |
| :--- | :--- | :--- |
| **Vert Forêt** (Primaire) | `#2E8B57` | Boutons d'action, Titres, Logo, Bordures actives |
| **Vert Feuille** (Secondaire) | `#3CB371` | Survol des boutons, Éléments de succès |
| **Or** (Accent) | `#FFD700` | Liens importants, Mises en avant, Étoiles de notation |
| **Gris Foncé** (Texte) | `#333333` | Texte principal, Pied de page |
| **Gris Clair** (Fond) | `#F4F4F4` | Arrière-plan général de l'application |
| **Blanc** (Cartes) | `#FFFFFF` | Fond des conteneurs, Cartes, Menus |

## 2. Typographie

Nous utilisons des polices modernes et lisibles issues de Google Fonts.

### Titres : **Outfit**
Utilisée pour tous les niveaux de titres (`h1`, `h2`, `h3`) et le logo.
*   **Famille** : Sans-serif
*   **Graisses** : 500 (Medium), 700 (Bold)
*   **Lien** : [Google Fonts - Outfit](https://fonts.google.com/specimen/Outfit)

### Corps de texte : **Inter**
Utilisée pour les paragraphes, boutons, et éléments de formulaire pour une lisibilité optimale.
*   **Famille** : Sans-serif
*   **Graisses** : 400 (Regular), 500 (Medium), 600 (Semi-Bold)
*   **Lien** : [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)

## 3. Composants UI (Interface Utilisateur)

### Boutons
*   **CTA Principal** : Fond `#2E8B57`, Texte Blanc, Coins arrondis (5px).
    *   *Hover* : Fond `#3CB371`, légère élévation.
*   **Liens** : Texte `#333`, souligné ou coloré au survol.

### Cartes (Cards)
*   Fond Blanc (`#FFFFFF`)
*   Ombre légère : `box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1)`
*   Padding interne : `1rem` ou plus.
*   Utilisées pour : Liste des trajets, Profils, Témoignages.

### Navigation
*   **Desktop** : Barre horizontale, Logo à gauche, Liens au centre, Connexion à droite.
*   **Mobile** : Les éléments s'empilent verticalement (Flexbox Column), centrage du contenu.
